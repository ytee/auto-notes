# Automotive Functional Safety — 10-Day Senior Preparation

Static Netlify-ready website with 100 questions covering FSM, HARA, SysML/MBSE, system architecture, brakes, ESC/control, requirements, FMEA/FTA/DFA, ASIL decomposition, hardware, software, CAN/OBD, V&V, suppliers, safety case and leadership.

## Run locally

```bash
cd automotive-functional-safety-prep
python3 -m http.server 8000
```

Open `http://localhost:8000`.

## Deploy to the existing Netlify site

1. Back up the current deployed app.
2. Unzip the package.
3. In Netlify, open the site and go to **Deploys**.
4. Drag the `automotive-functional-safety-prep` folder into the manual deploy area.

For Git deployment:

- Build command: leave blank
- Publish directory: `.`

## Features

- Ten daily 30–60 minute modules
- 100 conceptual, practical, scenario, review and leadership questions
- Topic tracks independent of the day sequence
- Domain-to-ISO interface matrix
- Job-description focus map
- Hidden answer frameworks and interviewer probes
- Search, filters, bookmarks, self-rating, completion progress and timer
- Browser-local progress; no backend required
- Print-friendly layout
